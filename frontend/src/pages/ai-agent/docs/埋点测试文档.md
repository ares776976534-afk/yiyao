## 测试说明

### 验证方法
1. 打开 Chrome DevTools (F12)
2. 切换到 Network 面板
3. 在 Filter 中输入 `aplus` 或 `log.mmstat`
4. 执行对应操作，观察是否有请求发出
5. 点击请求，查看 Payload 中的 `gokey` 参数是否包含期望的字段

### gokey 参数说明
- 参数值经过 URL 编码，中文会显示为 `%E5%95%86%E5%93%81` 这种形式
- 可以使用在线 URL 解码工具查看原始内容
- 或在 DevTools 中点击「查看 URL 编码」按钮

### 常见问题
1. **没有看到请求**: 检查组件是否正确传递了 `logKey` 属性
2. **gokey 缺少字段**: 检查 `log.record()` 调用时传递的参数对象
3. **cardSubType 不对**: 检查 FormatList 配置是否正确
